# Movie Recommendation System

This project is a **Content-Based Movie Recommendation System** built using **Machine Learning**.  
It recommends movies to users based on similarity between movie features like genre, keywords, cast, and overview.

---

## Dataset

The dataset is sourced from **Kaggle**:  
[Movie Dataset on Kaggle](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)

The dataset includes:  
- Movie titles  
- Genres  
- Keywords  
- Cast information  
- Movie overview  
- Ratings  

---

## Files

- **Jupyter Notebook (.ipynb)** – Contains the full implementation of the recommendation system, including data preprocessing, feature extraction, similarity calculation, and movie recommendation.  

---

## Features

- Content-based filtering using **Cosine Similarity**  
- Personalized movie recommendations  
- Handles movie metadata like genre, keywords, and cast  
- Fast and interactive output using **Streamlit** (optional deployment)
